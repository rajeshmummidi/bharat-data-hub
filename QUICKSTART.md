## ğŸ†˜ Troubleshooting

### Issue: Server shows error
**Solution:** 
- Check port 5000 is available
- Restart server: `node server.js`
- Clear browser cache and refresh

### Issue: Prices not changing
**Solution:**
- Wait a few seconds for next update (5-second interval)
- Check browser console for errors
- Verify server is running

### Issue: Can't execute trade
**Solution:**
- Check balance is sufficient
- Verify stock symbol is correct
- Check quantity is > 0
- See error message in notification

### Issue: Watchlist disappeared
**Solution:**
- Browser localStorage may be cleared
- Re-add stocks to watchlist
- Check browser settings allow localStorage

### Issue: Chart not showing
**Solution:**
- Refresh page
- Try different stock
- Check browser supports HTML5 Canvas
- Clear browser cache
